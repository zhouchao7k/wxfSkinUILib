WXF-SKINUI-LIB
======================
ui基础实现库，基础实现ui的解析和封装处理的操作


使用方法
----------------------


程序结构
----------------------
* 程序分三层
	1. BaseSkinUI
	2. SkinViewBlock
	3. SkinViewLogic


项目结构
----------------------

	/testing <- 测试代码
	/src     <- 实现代码
	/bin     <- swc输出目录
	/bin-debug <- 测试输出目录
	/build.xml <- 编译输出swc以供使用
	/build.properties <- 描述build.xml需要用到的变量
